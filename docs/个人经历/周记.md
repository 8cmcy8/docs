- 汇编语言

  看了三章的内容，记录新指令和注意的点

  - 标志寄存器  PSW

    - abc 和 sbb 用于进行更大位数的运算

    - cmp 相当于减法，不保存值，比较大小

    - 六个条件转移指令，满足条件就 jmp

    - 串传送指令 movsb、movsw + 两个 df 设置指令 cld（正）、std（负）

      rep 次数由 cx 寄存器中的值决定

    - pushf 和 popf 将标志寄存器的值压栈

  - 内中断

    - 8086 cpu 中断向量表存放位置 0000:03FF，最后 256 字节是空的

    - 中断处理程序编写 & iret指令

      在栈中弹出 CS:IP 的值以及弹出寄存器的值

      - 保存用到的寄存器
      - 处理中断
      - 恢复用到的寄存器
      - 用 iret 指令返回

    - 中断特殊情况：ss 和 sp 最好是连续设置
